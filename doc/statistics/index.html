第2章：1つの変数の記述統計

1つの変数の要約

  • 変数

    • 例えば、数学のテストの点数や辛いものの好き嫌い。これらは人によって違う値をとる。 これを統計学では『変数』と呼ぶ

  • 量的変数

    • データが数で表され、データの種類ではなく、その値の大小(数量)に意味がある
  • 質的変数
    • データを分類したもの(例: 男/女、好き/嫌い)を値とする。またその値(分類)をカテゴリという
    • 選択肢が2つの場合(男/女、好き/嫌い)は二値変数という

データの視覚的表現

  • 度数
    • 分類(カテゴリ)に含まれるデータの個数
    • 量的変数では、その値の範囲で分類する
      • 値が100未満、100以上200未満、200以上とか
    • 質的変数では、その値で分類する
      • 「好き」「嫌い」「どちらでもない」
1.test <- c(59, 74, 59, 74)

上記のcはcombine(結合)の略

  • 度数分布

    • 質的データや、量的データでも値が離散型のデータにおいて、そのデータの値とそれが現れる度数(頻度)をまとめたもの
      • 80名の学生の物理学の成績をS, A, B, C,Dの5ランクに分けたとき、それぞれのランクの人数の分布は度数分布といえる
    • 度数分布を表にしたものを度数分布表という

  • ヒストグラム

    • 度数分布から、横軸にデータの分類を、縦軸にその度数を表すグラフを書いたもの
    • Rではhistという関数で表示できる

代表値

分布の中心的位置を示す、そのデータを代表する値のこと

  • 平均値 (mean)
  • 中央値 (median)
  • 最頻値 (most frequent value)
    • 平均値や中央値が計算出来ない場合に、最頻値を使う
    • つまり、質的変数の場合に最頻値を代表値として使う

散布度

  • 変数の値の散らばりのこと
  • どんなデータでも、平均だけではその特徴はわからない
  • 平均のまわりにどのようにデータが散らばっているかも、データも性質を考える上でとても重要な情報

主な散布度

  • 分散
  • 標準偏差
  • 平均偏差
  • 範囲(レンジ)

分散

  • 散布度を表すのによく使われる

  • データn-平均

    • 平均からの偏差
    • 各データの値から平均を引いたもの

  • 標本分散と不偏分散の違い

    • 標本分散
      • 手元にあるデータの「散らばり」を表す値
    • 不偏分散
      • 手元にあるデータは何らかの大きなデータの集まり(母集団)から抽出したもの
      • 母集団のデータの散らばりを推定した値
      • Rでは不偏分散を計算するための関数varが用意されている

  • 平均からの偏差

    • 各データの値から平均を引いたもの
    • データ - 平均

標準偏差(standard deviation)

  • 散布度の指標として最も使われる
  • この値が大きいほどデータの散らばりも大きい
  • (不偏)分散の正の平方根の値
  • Rではsd関数が用意されている

平均偏差

  • 「平均からの偏差」の絶対値の平均
  • 平均からどの程度ずれているかを表す
  • mean(abs(テスト結果 – mean(テスト結果)))
  • abs関数は絶対値を求められる

範囲(レンジ)

  • データの最大値と最小値の差のこと

母集団と標本

  • 母集団
    • 推測対象の要素の集まり（おおまかなにいうと）
  • 無限母集団
    • 母集団の要素の個数を無限に推測出来る場合
    • 母集団の要素数が多かったり、実際には存在していないものだったりすることがあるので、母集団をそのまま扱うことは一般に難しい
  • 標本(サンプル)
    • 実際に手元にデータ
    • 標本は母集団の一部
    • 統計学は、この標本(サンプル)の性質を調べることによって、母集団の性質を予測したり、比較したりする手段を与えてくれる

標準化

平均と標準偏差がある特定の値になるように、全てのデータの値を同じ式を使って変換すること

標準得点

  • 標準化された得点のこと
  • 特に平均0、標準偏差1になるように標準化したもの

偏差値

  • 標準化された得点のこと
  • 平均50、標準偏差10になるように標準化したもの

関数のまとめ

目的 関数名と書式 使い方
関数分布(表)の作成 table(データ) table(指導法)
ヒストグラムの描画 hist(データ) table(心理学テスト)

データの個数 length(データ) length(心理学テスト)

中央値を求める median(データ) median(心理学テスト)
平方根を求める sqrt(データ) sqrt(2.0)

絶対値を求める abs(データ) abs(-3.5)
最大値を求める max(データ) max(心理学テスト)
最小値を求める min(データ) min(心理学テスト)

第3章：2つの変数の記述統計

2つの変数の関係

  • 相関
    • 2つの量的変数の関係のこと
  • 連関
    • 2つの質的変数の関係のこと

量的変数の関係(相関)を図で表す: 散布図

  • 散布図

    • 2つの対応のあるデータを2次元の平面上にプロットして得られる図
    • Rではplot関数で散布図を表示できる

  • 正の相関

    • xの値が増加するとyの値が増加する関係
  • 負の相関
    • xの値が増加するとyの値が減少する関係
  • 無相関
    • x,yに関係がない場合

量的変数の関係(相関)を数値で表す: 共分散、 相関係数

  • 偏差

    • それぞれのデータとその平均の差
    • 「平均からの偏差」のこと

  • 共分散Sxy

    • 偏差の積の平均
    • データ同士の関係性を見れる
      • ただし、比較同士の単位が違うと比較が出来ない
        • 変数xの平均
        • 変数yの平均
    • Rではcov関数で共分散を求められる
      • cov関数では、分母がnではなく、 n-1になっている
      • そのため不偏共分散と呼ぶことがある

  • 相関係数

    • 相関の度合いを数値で表す
    • よく使われるのがビアンソンの相関係数
    • Rではcor関数で相関係数を求められる
  • ビアンソンの相関係数
        • 変数xの標準偏差
        • 変数yの標準偏差
    • 相関係数rは-1≦r≦1の範囲の値をとる

相関の評価

相関係数 大きさの評価
-0.2 ≦ r ≦ 0.2 大ほとんど相関なし
-0.4≦r < -0.2 0.2 < r ≦ 0.4 弱い相関あり
-0.7≦r < -0.4 0.4 < r ≦ 0.7 中程度の相関あり
-1.0≦r < -0.7 0.7 < r ≦ 1.0 強い相関あり

    • yとxに強い相関がある時に1次方程式で表すことが出来る

  • 単回帰分析

質的変数の関係(連関)を表す: クロス集計表、ファイ係数

  • クロス集計表

    • 質的変数同士の関係を見るのに使う

  • ファイ係数

    • 質的変数に対する「特殊な相関係数」
    • 1と0からなる変数（二値変数）で計算する相関係数
    • データを’1’,’0’に変換し、相関係数(cor)で求められる

関数のまとめ

目的 関数名と書式 使い方
散布図を描く plot(x, y) plot(数学テスト, 　英語テスト)
共分散を求める cov(x, y) cov(数学テスト, 　英語テスト)
相関係数を求める cor(x, y) cor(数学テスト, 　英語テスト)
単回帰分析を行う lm(y ~ x) lm(英語テスト ~ 数学テスト)
クロス集計表を書く table(x, y) cor(数学テスト, 　英語テスト)
場合分けをする ifelse(条件, 真の場合, 偽の場合) ifelse(統計学==”好き”, 1, 0)

第4章：母集団と標本

母集団と標本の関係

  • 母集団
    • 対象のデータ全体
  • 標本
    • 母集団から一部のデータを抽出したもの
  • 標本抽出
    • 母集団から標本を抽出すること
  • 母数
    • 母集団の性質をあらわす統計的指数（比率、平均、分散、相関係数など）

推測統計

  • 手元にある標本から母集団の性質（母数）を推測すること
  • 推定と検定の分けられる
  • 推定
    • 点推定
      • 母数の値に対し具体的な値を求める
      • 一つの値で推定の結果を表すもの
    • 区間推定
      • ある幅を持った区間で結果を表すもの

推定の基礎

  • 標本統計量
    • 標本データから計算される「平均」「分散」「標準偏差」「相関係数」などの値
  • 推定値
    • 標本データを用いて計算した母数の推定量
目的 関数名と書式 使い方
母平均 標本平均 mean
母分散 不偏分散 var
母標準偏差 不偏分散の正の平方根 sd
母相関係数 標本相関係数 cor
母比率 標本比率 prop.test

標本抽出にともなう誤差

  • 標本誤差
    • 母数の値と推定値とのズレ
    • データをたくさん集めれば、生じうる誤差は小さくできる
  • 標本分布
    • どのような推定値が得られる可能性があるのか、についての手がかり

推定値の信頼性を調べる方法

  1. 標本抽出の方法
    • 単純無作為抽出
      • 母集団の中のどのデータにも選ばれる可能性をもっているような標本抽出の方法
      • 「結果」については問わない
        • たまたま偏った標本になる可能性も織り込み済みで状態で理論が作られている
    • 無作為標本
      • 単純無作為抽出によって得られた標本
  2. 単純無作為抽出によって得られるデータの性質としての確率変数
    • 確率変数
      • 実際の結果が得られるまでどのような値か決まっていない変数
      • 例：サイコロを降った時の出目
        • 同じ手順でデータを得ても結果の再現性がない
        • 単純無作為抽出によって得られるデータは確立変数といえる
  3. 確率変数がどのような値を取るかを示す確立分布
    • 確率変数の実現値
      • 抽出の結果、確率変数がとる値
    • 確率分布
      • 確立変数がどのような値をどのような確率で取るかということを表した分布
      • 確率分布は実際に得られたデータから作成するものではなく、理論的に決められるもの
      • ある確率変数Xの実現値がある確率分布Aに基づいて決まる場合
        • 「確率変数Xは確率分布Aにしたがう」といういい方をする
        • 「X~A」と一般的に表す
  4. 確率分布を用いた母集団の表現としての母集団分布
    • 母集団分布
      • ある変数の母集団における分布
      • 無作為抽出の場合、標本データの確立分布は母集団分布と同じになる
  5. 代表的な母集団である正規分布
  6. Rを使って正規分布の母集団から標本を抽出する方法

正規分布

正規分布の表現方法

μは平均値、σ^2は分散 (σは標準偏差)

  • 標準正規分布
    • 平均0 分散1の正規分布 — N(0,1)
  • 離散変数
    • 整数など、とびとびの値をとる変数
    • 例）サイコロの出目とか
  • 連続変数
    • 実数値をとる変数
    • 例）正規分布に従う確率変数
  • 確率密度
    • 正規分布の図の縦軸は確率密度
    • 確率＝確率密度*確率変数の範囲

  • 確率密度関数

    • 確率密度を確率変数の値の関数として表したもの

  • 正規分布の性質

    • μ-σからμ+σ の範囲(偏差値で言えば、40点から60点の範囲)の確率はおよそ 0.683
    • μ-2σからμ+2σ の範囲(偏差値で言えば、30点から70点の範囲)の確率はおよそ 0.954
    • μ-3σからμ+3σ の範囲(偏差値で言えば、20点から80点の範囲)の確率はおよそ 0.997
正規分布の標準偏差

平均点から変曲点（上にふくらんだ部分と下にへこんだ部分の境目）までの距離を表す

標本分布

  • 標本統計量(標本平均、標本分散など)に関する確率分布のこと
  • 標本における個々のデータの実現値を表した度数分布ではなく、標本統計量の確率分布
  • 母集団分布と標本統計量の種類、そしてサンプルサイズが決まると理論的（数学的）に導かれる。 実際のデータから作成されるわけではない
標本分布から何がわかるのか
  • どれくらいあてになる推定値が得られる可能性があるかを知ることができる
  • 標本分布を調べる時の観点
    • 標本分布が母数の本当の値の中心として分布しているか → 平均
      • これが満たされないと推定値と母数の誤差が大きくなる
    • 標本分布が横に大きく広がっていないか → 標準偏差
      • 推定値にどの程度誤差が生じるか調べるもの
標本分布を「経験的」に求める
  • 「経験的」
    • 「現実の得られたデータに基づいて」という意味
不偏性
  • ある推定量の標本分布の平均が、推定しようとしている母数の値と一致すること
    • 10000個の推定値の平均値を求めると、実際の母平均の値とほぼ一致する
    • 無限に推定を繰り返すと、その推定値（標本平均）の平均（標本分布の平均）は母平均と一致する
  • 不偏性のある推定量のことを不偏推定量という
標準誤差
  • 推定量の標本分布の標準偏差のこと
  • 標本分布の横への広がりのこと
  • 標準誤差が少ない＝標本分布の広がりが小さい＝ばらつきが少ない

標本平均以外の標本分布

標本分散と不偏分散の違い
  • 標本分散
  • 不偏分散
    • 不偏性を持つという点では望ましい性質を持つ
    • しかし、標本分散に比べて過大な推定値が得られる可能性が高い
      • 当たりハズレが大きい
    • 不偏分散の平方根は母標準偏差の不偏推定量ではない
目的 関数名と書式 使い方
小数点以下を切り上げ ceiling(a) ceiling(1.5) ⇒ 2
一様分布に従う乱数の発生 runif(n,min,max) runif(n=10, min=0, max=6) ⇒
0から6までの範囲の一様乱数を10個生成
決まったパターンの乱数の生成(乱数の『種』の指定) set.seed(seed) set.seed(1) ⇒
乱数の種を1に設定。これを指定すると常に同じパターンの乱数が生成される
棒グラフの作成 barplot(height,names.arg) barplot(c(2/3, 1/3), names.arg=c(“男性”,”女性”))
⇒ 高さ2/3, 1/3の棒に『男性』『女性』のラベルをつける
関数の曲線を描く curve(func, from, to) curve(dnorm(x), from=-3, to=3)
⇒ 標準正規分布を描く
正規分布の確率密度関数 dnorm(x,mean,sd) dnorm(0,mean=0,sd=1)
⇒ 標準正規分布におけるx=0のときの確率密度
正規分布に従う乱数の発生 rnorm(n,mean,sd) rnorm(n=10,mean=50,sd=10)
⇒ N(50,102)に従う乱数を10個生成
数値を格納する場所の確保 numeric(length) a <- numeric(length=10000)
⇒ 変数aに数値データを10,000個格納できるようにする
連続データを作る a:b x <- 0:20
⇒ 0から20までの整数の連続データを作り変数xに格納
同じ処理の繰り返し for(i in データ) { 処理 } for(i in 1:100) { cat(“hello, world!\n”) }
⇒ { }内の処理を100回繰り返す: hello, world!が100回表示される

第5章：統計的仮説検定

統計的仮説検定の必要性

  • 統計的仮説検定
    • 限られた標本データから、確率論に基づいて、主張を行うための方法

統計的仮説検定の手順と用語

検定の手順

  1. 母集団に関する帰無仮説と対立仮説（両側or片側検定）を設定
  2. 検定統計量を選ぶ
  3. 有意水準αの値を決める
  4. (データを収集した後)データから検定統計量の実現値を求める
  5. 検定統計量の実現値が棄却域に入れば、帰無仮説を棄却して、対立仮説を採択する。棄却域に入らなければ、帰無仮説を採択する。
1. 帰無仮説と対立仮説

統計的仮説検定では「差がない」という帰無仮説と「差がある」という対立仮説を立てる

  • 帰無仮説
    • 本来主張したい逆の仮説
    • この仮説が棄却されることを目的に仮説検定を行う
  • 対立仮説
    • 本来主張したい仮説

対立仮説の設定により、検定は次のどちらかで行う(通常は両側検定で行う※より厳しい条件なので)

  • 片側検定
    • 対立仮説が下記の場合
      • 母平均 μ>0 (もしくは μ<0)
      • 母相関係数 ρ>0 (もしくは ρ<0)
      • 母平均の差 μ1>μ2 (もしくは μ1<μ2)
  • 両側検定
    • 対立仮説が下記の場合
      • 母平均 μ≠0
      • 母相関係数 ρ≠0
      • 母平均の差 μ1-μ2≠0

要するに、両側検定では、例えば母平均μ≠0を調べるには、母平均μ > 0 と μ < 0 の両方を調べなければならない

2. 検定統計量
  • 検定統計量
    • 統計的仮説検定のために用いられる標本統計量のこと
    • 代表的な検定統計量の例
      • ｔ, x^2, F
  • 検定統計量の実現値
    • 実際のデータ(手に入った標本)を基にして計算して得られる具体的な値のこと
    • 帰無仮説にそぐわず、対立仮説に合っているほど0から離れた値を示す
3. 有意水準と棄却域
  • 対立仮説を採択する基準が有意水準(αで示される)
  • 有意水準は5%または1%(α=0.05またはα=0.01)に設定することが多い
    • つまり標本が100回に5回以下しか現れないデータ
    • こんなことは偶然では起こり得ない
    • だから、帰無仮説が成り立たない
  • 帰無仮説が正しいものとして考えた時の標本分布を帰無分布という
    • 帰無分布に基づいて確率計算される
  • 棄却域
    • 帰無仮説のもとで、非常に生じにくい検定統計量の値の範囲のこと
    • 採択域
      • 棄却域以外の部分
      • 帰無仮説が採択される領域
    • 臨界値
      • 棄却域と採択域の境目の値
4&5. 統計的仮説検定の結果の報告
  • 検定統計量の実現値が
    • 棄却域に入った場合
      • 「差がない」という帰無仮説を棄却し、「差がある」という対立仮説を採択する
    • 棄却域に入らない場合
      • 「検定の結果、有意の差がなかった」または「有意差が認められなかった」と書く

p値

  • 帰無仮説が正しいという仮定のもとで、標本から計算した検定統計量の実現値以上の値が得られる確率
  • p値が有意水準より小さい時に帰無仮説を棄却する

第1種の誤りと第2種の誤り
  • 第1種の誤り α
    • 「帰無仮説が真のとき、これを棄却してしまう」
    • この種の誤りを犯す確率が「有意水準」または「危険率」
  • 第2種の誤り β
    • 「帰無仮説が偽のとき、これを採択してしまう（棄却できない）」
    • 本当は差があるのに「差がない」と判断してしまう誤り
